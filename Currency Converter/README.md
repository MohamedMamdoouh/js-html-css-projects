# Currency Converter

A simple currency conversion app built with HTML, CSS, and JavaScript. The tool converts between currencies using preset exchange rates or input values.

## Summary

Enter an amount and select currencies to convert between and view the converted total.

## Features

- Amount input
- Currency selection
- JavaScript conversion logic
- Clean interface with visual currency elements

## Usage

1. Open `index.html` in a browser.
2. Enter an amount.
3. Choose the source and target currency.
4. View the conversion result.
