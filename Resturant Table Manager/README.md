# Restaurant Table Manager

A restaurant table management UI that helps simulate seating and table organization. This frontend project is built with HTML, CSS, and JavaScript, and includes print-friendly styling.

## Summary

Use the app to manage table layout, reservations, or seating status and preview the restaurant interface in a styled format.

## Features

- Table manager layout
- Seating and reservation UI
- Print-friendly style sheet

## Usage

1. Open `index.html` in a browser.
2. Use the interface to manage tables or reservations.
3. Print the view if needed using the print stylesheet.
