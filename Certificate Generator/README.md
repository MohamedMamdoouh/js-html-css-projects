# Certificate Generator

A creative frontend project for generating customizable certificates using HTML, CSS, and JavaScript. The app lets users enter details and preview a certificate design.

## Summary

Fill in the recipient name, course or achievement, and date to generate a certificate layout.

## Features

- Certificate text input fields
- Preview of generated certificate
- Styled certificate template with decorative graphics

## Usage

1. Open `src/index.html` in a browser.
2. Enter the required certificate details.
3. Preview the certificate on screen.
