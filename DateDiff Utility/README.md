# DateDiff Utility

A frontend utility that calculates the difference between two dates. Built with HTML, CSS, and JavaScript, it shows the elapsed time in days, months, and years.

## Summary

Enter two dates and the app displays how much time has passed between them.

## Features

- Start and end date inputs
- Difference calculation in readable units
- Simple, responsive layout

## Usage

1. Open `index.html` in a browser.
2. Select the start and end dates.
3. Submit to see the date difference.
