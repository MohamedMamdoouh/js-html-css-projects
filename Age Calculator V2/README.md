# Age Calculator V2

A second version of the Age Calculator with enhanced UI and improved validation. This project demonstrates similar date calculations with a refreshed user experience.

## Summary

Enter your birth date and the app computes your age. This version may include better styling and input handling.

## Features

- Birthdate input form
- Improved validation and user feedback
- JavaScript-based age calculation

## Usage

1. Open `index.html` in a browser.
2. Enter your date of birth.
3. Submit the form to view the age result.
