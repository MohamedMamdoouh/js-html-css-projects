# Subscription Expiration

A small frontend app for tracking subscription dates and expiration status. Built with HTML, CSS, and JavaScript, it helps calculate when a subscription will expire.

## Summary

Enter subscription details and view the scheduled expiration or remaining time until renewal.

## Features

- Subscription date input
- Expiration calculation
- Visual status display

## Usage

1. Open `index.html` in a browser.
2. Enter subscription details.
3. Review the expiration status.
