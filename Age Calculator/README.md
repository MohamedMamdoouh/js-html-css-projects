# Age Calculator

A simple frontend utility that calculates a person’s age based on their birth date. The project uses HTML forms, CSS styling, and JavaScript date calculations.

## Summary

Enter your date of birth and the app displays the calculated age in years, months, and days.

## Features

- Birthdate input form
- Age calculation using JavaScript
- Clean responsive layout

## Usage

1. Open `index.html` in a browser.
2. Enter your date of birth.
3. Submit the form to see the calculated age.
