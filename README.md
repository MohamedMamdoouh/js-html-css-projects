# JS, HTML, CSS Projects

A collection of frontend projects built with plain HTML, CSS, and JavaScript. These projects demonstrate UI design, interactivity, DOM manipulation, form handling, and small utility applications.

## Included Projects

- Age Calculator
- Age Calculator V2
- Certificate Generator
- Currency Converter
- DateDiff Utility
- Restaurant Table Manager
- Simple E-Commerce
- Student CRUD Operations
- Subscription Expiration
- Task Manager
- Weather App

## How to Use

1. Open the project folder you want to view.
2. Open the `index.html` file in a browser.
3. If the project has multiple files or folders, open the main HTML file for that project.

## Recommended Workflow

- Use a modern browser such as Chrome, Edge, or Firefox.
- If you want live reload while developing, use a simple local server such as VS Code Live Server or Python's `http.server`.

## Notes

- All projects are client-side only and do not require a backend.
- Assets and styles are stored inside each project folder.
- Some projects may be organized with nested HTML/CSS/JS files.
