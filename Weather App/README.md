# Weather App

A frontend weather app interface that displays weather information using HTML, CSS, and JavaScript.

## Summary

Search for weather conditions and view results in a polished weather dashboard layout. It uses api calls to fetch weather data and displays it in a user-friendly format.

## Features

- Weather search input
- Display of current weather conditions
- Responsive weather UI

## Usage

1. Open `index.html` in a browser.
2. Enter a location or city.
3. View the weather results.
