# Task Manager

A frontend to-do list application built with HTML, CSS, and JavaScript. The app supports adding, completing, and removing tasks.

## Summary

Organize tasks in a simple task manager interface and manage your daily checklist.

## Features

- Add new tasks
- Mark tasks as complete
- Remove completed or unwanted tasks
- Interactive task list UI

## Usage

1. Open `index.html` in a browser.
2. Add tasks using the input form.
3. Mark tasks complete or delete them.
