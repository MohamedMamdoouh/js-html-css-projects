# Student CRUD Operations

A frontend application demonstrating Create, Read, Update, and Delete operations for student records using HTML, CSS, and JavaScript.

## Summary

Manage a student list directly in the browser, with form-based entry and in-place editing.

## Features

- Add new student records
- Edit existing student entries
- Delete students from the list
- Dynamic UI updates without page reloads

## Usage

1. Open `index.html` in a browser.
2. Use the form to add student details.
3. Edit or delete records as needed.
