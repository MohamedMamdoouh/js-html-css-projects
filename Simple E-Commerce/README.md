# Simple E-Commerce

A frontend e-commerce prototype with separate pages for authentication, shopping, and checkout. The project is built with plain HTML, CSS, and JavaScript and uses a simple in-browser data structure for demo shopping behavior.

## Summary

Browse products, add items to a cart, and proceed through a checkout flow in a static frontend storefront.

## Features

- Auth page setup
- Shop page with products
- Checkout page flow
- JavaScript-driven cart behavior

## Usage

1. Open `Auth/auth.html` in a browser to start the app.
2. Navigate to the shop and add items to the cart.
3. Open `Checkout/checkout.html` to complete the demo checkout.
